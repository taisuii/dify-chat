# 标注

> 最小可用版本：v0.7.0。

在 Platform 后台，支持对应用进行标注管理：

![标注管理操作](/guide__platform_annotation_manage_button.png)

![标注管理操作](/guide__platform_annotation_manage_drawer.png)

同时，也可以允许用户在前端界面中进行 AI 回复标注。要启用该功能，需要在应用配置中将 "允许标注" 设置为 "启用"（默认为禁用）：

![是否允许用户标注](/guide_platform_annotation_config.png)

选择启用并保存之后，你可以在对应应用的 AI 回复消息下方，看到标注按钮：

![标注按钮](/guide__react_annotation_button.png)

点击标注按钮，将会展示标注创建抽屉，你可以编辑对应的问题和回复：

![标注创建抽屉](/guide__react_annotation_drawer.png)

用户提交之后，你就可以在 Platform 后台看到标注记录了。
